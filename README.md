# Minecraft-DOS
Minecraft Launcher in command prompt
need python
# Usage
----------------------------------------------------
listver (None)
List all version
----------------------------------------------------
download (Vers)
Download a version
----------------------------------------------------
launch (Vers)
Launch a version
----------------------------------------------------
config (None)
Configure your profile
----------------------------------------------------
clear (None)
Clear Console
----------------------------------------------------
refrver or F5 (None)
Refresh installed versions
----------------------------------------------------
mslogin (None)
Microsoft Login
----------------------------------------------------
authlib (None)
Authlib-Injector Yggdrasil Sign
----------------------------------------------------
alaunch (Vers)
Authlib-Injector Launch
----------------------------------------------------
mslaunch (Vers)
Microsoft Account Launch
----------------------------------------------------
modsmenu (None)(Beta)
Display available mods from CurseForge API
----------------------------------------------------
checkdisk (None)
Check available disk space
----------------------------------------------------
sysinfo (None)
Show system information
----------------------------------------------------
uninstall (Vers)
Uninstall a Minecraft version
----------------------------------------------------
downmod (ModName GameVersion)(Beta)
Download a mod from CurseForge
----------------------------------------------------
